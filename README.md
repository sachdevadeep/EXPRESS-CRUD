# Express CRUD API

A simple REST API built using Node.js and Express.js.

## Features

- Get all users
- Get user by ID
- Create new user
- Update user
- Delete user

## Technologies Used

- Node.js
- Express.js

## Installation

```bash
npm install
npm start
```

Server runs on:

```
http://localhost:8000
```

## API Endpoints

GET /api/users

GET /api/users/:id

POST /api/users

PATCH /api/users/:id

DELETE /api/users/:id
